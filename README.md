# Weather-Dashboard

## Introduction:

```
when user search for a city the current and future weather conditions for
that city will display as following and that city is added to the search history:
- City name
- Date
- An icon representation of weather conditions
- Temperature
- Humidity
- Wind speed
- UV index
- 5-day forecast
```
